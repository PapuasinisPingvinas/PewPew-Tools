# PewPew Tools
A tool that let's you install and update all PewPew tools in one place.
## Installation
Load PowerShell script with the following command:
```powershell
iwr -useb "https://raw.githubusercontent.com/PapuasinisPingvinas/PewPew-Tools/master/PewPewTools.ps1" | iex
```
## Info:
* MidiToPewPew currently doesn't install.
* DO NOT REMOVE OR REPLACE `config.json` FILE AS IT IS USED FOR REMOVING AND UPDATING.
* VSCode extension and Blender Add-On doesn't install, it only downloads them. Please use VSCode and Blender respectively to install them.
* All files are in `<YourUser>\PewPewTools` directory.
